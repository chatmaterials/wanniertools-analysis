# Model Handoff Guidance

Load this file when the question is whether a Wannier model is ready for WannierTools.

## Preconditions

- the Wannier model should already reproduce the parent bands in the target energy window
- the seedname and file naming should be consistent
- the occupied-band count must be known and justified

## Surface setup

- choose the surface orientation deliberately
- state whether the analysis is exploratory or intended for publication-quality interpretation
- if the user cannot justify the surface choice, recommend checking the crystallographic context first
